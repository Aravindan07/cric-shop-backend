## CricShop Backend

This repository contains **API'S** for [this](https://cric-shop.netlify.app/) application.

The List of **API'S** are as follows,

-   Product Listing API.
-   Product Posting API.
-   Add to Cartlist API.
-   Add to Wishlist API.
-   Remove from Wishlist.
-   Remove from cartlist.
